# Sečti n čísel
Napiš program, který sečte `n` čísel.


## Typ vstupu

Na vstupu programu dostaneš číslo `n` a dále dostaneš `n` čísel (na samostatných řádcích)

## Příklady
### Příklad 1
#### Vstup
```
2
1
11
```

#### Výstup
```
12
```
---
### Příklad 2
#### Vstup
```
4
10
0
-2
0
```

#### Výstup
```
8
```
---
### Příklad 3
#### Vstup
```
1
0
```

#### Výstup
```
0
```
